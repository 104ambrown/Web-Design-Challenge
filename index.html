The Dirty Deets

The purpose of this project was to examine how changes in latitude impact the weather experienced in a location. Data from 500 cities was collected by using OpenWeatherMap API. The data was then cleaned and entered into a dataframe.

The gathered, cleaned, and compiled data was then used to generate graphs using Matplotlib to allow a visual analysis of the data to be performed. Weather elements such as humidity, cloud cover, wind speed, and temperature were all considered. Analyses and conclusions can be found, (along with the plots), on each of their respective pages.
